OVERVIEW
HOPMON is a 3D arcade action game. There are 45 levels in total and to complete each level, Hopmon will have to collect all the crystals and place them on the warp zone where the star symbol is printed. 

You can continue play on the same level indefinitely. The game has also an automatic save feature so that you can continue playing the level you played last.

This game is freeware.

SYSTEM REQUIREMENTS
  • Windows 8/10/11
  • 32MB RAM
  • 10MB of free hard disk space
  • 3D accelerated Video Card
  • Joystick / Gamepad recommended
KEY CONTROLS
Arrow Keys Move Hopmon
[Shift] or [Z] Key Fire
[Ctrl] or [X] Key + Arrow Keys Move Camera Viewpoint
[P] Key Pause
[Esc] Key Back or Exit
This game supports Joystick / Gamepad.
